# Privacy Policy

## DynaTools
Last updated: 2026-02-19

DynaTools is a browser extension for Microsoft Dynamics 365 Finance & Operations environments.

## Data Collection
- DynaTools does not collect personal data.
- DynaTools does not transmit browsing data to external servers.
- DynaTools does not use analytics or tracking services by default.

## Data Storage
- DynaTools stores user preferences in browser extension storage (`chrome.storage.sync`), including:
- UI settings (toasts, environment strip, badge size).
- Environment classification rules/colors.
- Language menu definitions.
- Stored data remains in the user browser profile/sync account.

## Permissions Usage
- `storage`: save extension settings.
- `tabs` and `activeTab`: run toolbar actions for the active Dynamics tab.
- `downloads`: save screenshots initiated by the user.
- Host access: `*://*.dynamics.com/*` only.

## Third Parties
- DynaTools does not embed third-party scripts from external CDNs.
- DynaTools does not sell or share user data.
